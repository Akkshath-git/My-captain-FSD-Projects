# Pulse — Smart Fitness Companion Landing Page

Pulse is a smart fitness companion that reads your heart rate and recovery
data in real time and automatically adjusts your workout's intensity as you
train — instead of following a fixed plan that ignores how your body
actually feels that day. Built for runners, lifters, and everyday athletes
who train consistently but don't have a coach monitoring their vitals.

This is Mini Project 1 for the My Captain Full Stack Web Development course.

## 🔗 Live Demo
https://Akkshath-git.github.io/pulse-landing-page/

## 🛠️ Built With
- HTML5
- CSS3 (custom design tokens, animations)
- Bootstrap 5

## ✨ Features
- Responsive layout (mobile, tablet, desktop)
- Animated pulse-line SVG graphic in the hero section
- Live count-up statistic using `IntersectionObserver`
- Adaptive intensity & recovery-aware scheduling feature highlights
- Use-case section for Runner, Lifter, and Beginner personas
